Projects and Deployments

Un proyecto en Vercel es la representación de tu aplicación. Cada vez que realizas un cambio, se crea un despliegue, que es una versión única de tu app.

Los despliegues pueden ser de preview (para pruebas y revisiones) o de producción (la versión pública). Esto permite flujos CI/CD automatizados.

Use a Template

Comienza con rapidez usando plantillas preconfiguradas para frameworks como Next.js, React, Vue y más. Estas plantillas incluyen configuraciones óptimas de construcción y despliegue.

Ideal para principiantes que quieren probar Vercel en minutos.

Import Existing Project

Si ya tienes un proyecto en GitHub, GitLab o Bitbucket, puedes importarlo fácilmente a Vercel. Vercel detecta automáticamente el framework y configura el proceso de build.

Además, permite configurar variables de entorno y rutas personalizadas.

Add a Domain

Asigna un dominio personalizado a tu proyecto. Esto incluye subdominios, configuración automática de HTTPS y redirecciones.

Puedes usar registros DNS como A o CNAME para enlazarlo a tu proyecto.

Buy a Domain

Compra un dominio directamente en Vercel sin necesidad de recurrir a terceros. El dominio se integra automáticamente con tus proyectos.

Además incluye SSL gratis y administración simplificada desde el panel de Vercel.

Transfer an Existing Domain

Transfiere un dominio que ya posees a Vercel para gestionarlo directamente desde la plataforma. Requiere un código de autorización (Auth Code).

Una vez transferido, podrás beneficiarte de la integración nativa con proyectos y certificados automáticos.

Collaborate

Invita a otros desarrolladores a tu proyecto. Los roles definen permisos como Owner, Developer o Viewer.

Esto facilita flujos de trabajo en equipo, revisiones de despliegue y coordinación en grandes proyectos.

Next Steps

Una vez dominados los básicos, puedes explorar funcionalidades avanzadas como:

  • Funciones Serverless
  • Optimización de imágenes y assets
  • Analytics integrados
  • Integraciones con CMS y APIs
